# ❄️ Snowball Showdown

Hi! 👋

This is my game called **Snowball Showdown**.

In this game, two players have a snowball fight and answer science questions. If you get the answer right, your snowball become stronger! Bigger snowballs do more damage.

I made this game with **HTML, CSS and JavaScript**. It was fun but sometime hard.

---

## 🎮 How to Play

Player 1

- A = Move Left
- D = Move Right
- W = Change Snowball
- S = Throw Snowball
- 1 2 3 4 = Answer Questions

Player 2

- ← = Move Left
- → = Move Right
- ↓ = Change Snowball
- ↑ = Throw Snowball
- 7 8 9 0 = Answer Questions

---

## ⭐ Game Rules

- Two players play together.
- Answer the science question.
- Right answer = stronger snowball.
- Throw snowballs to hit your friend.
- First player to make the other player have 0 health wins!

Dont worry if you lose. Just play again 😄
